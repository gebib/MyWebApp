ABOUT ME

My name is Gebi Beshir. I’m an intermediate software developer, graduated with a bachelor's degree, as a software engineer from OsloMet, and have been learning and working as a freelancer since then.

I have a genuine interest in computers, technology and electronics, and therefore also studied “Dataelektronikk” at Årstad VGS, previous to my study of "Dataingeniør" at OsloMet. Having an electronic background also helped me to better understand electronics and computers in general, and how they operate, which have also been a great baseline for me, in order to be a better programmer.

While working as a freelancer, I have developed web applications using AngularJS. And tried out many personal and school related learning projects also with ReactJS, .NET, Java, and as well as developing website without JavaScript framework, such as this website, which is developed entirely with HTML, CSS and JavaScript.

I’ve yet to specialize in any specific area within software development, or as a fullstack developer, which is also what I’m looking forward to, but I'm currently able to work with an in depth software development for desktop and mobile software using Java or .NET, and JavaScript frameworks.
